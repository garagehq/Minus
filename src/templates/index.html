Detection Settings

VLM (FastVLM-1.5B)
Status: Loading...

Disabling VLM unloads the model from the Axera NPU to free memory. Re-enabling takes ~13 seconds to load.

Preload VLM at Startup

When enabled, VLM loads immediately at startup (even without HDMI signal). When disabled, VLM waits for HDMI signal before loading. Takes effect on next restart.

Blocking Optimizations

Block-duration Falloff

Minimum blocking time shrinks 3.0s → 1.0s as consecutive ads fire (floor 1.5s when OCR+VLM both agree). Lets the block end right as the ad does instead of holding for a full 3s every time.

HDMI Reconnect Grace (90s)

After the TV reconnects, suppress ad blocking for 90 seconds so you can navigate menus without overlays jumping in.

Greyscale Ad Preview

Desaturates the corner preview of the blocked ad so the Spanish overlay (bright, colorful) is more appealing than the ad.

IR Remote (HDMI Switch)

Controls the REI 8K HDMI switch via IR (header pin 38, PWM3). Lets autonomous mode rotate inputs to vary training data.

Status LEDs

8 WS2812B LEDs on header pin 19 indicate device state (initializing / idle / blocking / paused / no-signal / autonomous / wifi / error). Requires the SPI0-M2 overlay enabled.

Only when display is on

When on (default), the strip stays dark while the HDMI-TX display is disconnected or powered off. The state machine still runs — animations resume seamlessly when the display comes back.
